# Kardtest 2000

Многоканальный имитатор электрокардиосигналов для поверки электрокардиографов в соответствии с российскими и международными стандартами.

## Технические характеристики

### Каналы генерации (9 каналов: L, R, F, C1-C6)
* **Синусоидальные сигналы**: ± 5 мВ с шагом 0.1 мВ, от 0.01 до 100 Гц (метод **DDS**).
* **Прямоугольные сигналы**: ± 5 мВ с шагом 0.1 мВ, от 0.01 до 100 Гц (метод **DDS**).
* **Треугольные сигналы**: ± 5 мВ с шагом 0.1 мВ, от 0.01 до 100 Гц (метод **DDS**).
* **Белый шум:** реализован на базе алгоритма **Galois LFSR** (± 5 мВ).
* **Произвольные сигналы:** воспроизведение реальных ЭКС из Flash-памяти.

### Специальные тестовые выходы
* **Канал 10 (S):** синусоидальный сигнал 50 Гц (± 20 В, шаг 1 В) для проверки подавления синфазных помех.
* **Канал 11 (C):** постоянное напряжение ± 300 мВ (шаг 1 мВ) для проверки устойчивости к дрейфу изолинии.

## Аппаратная часть
* **Микроконтроллер:** ATMega162.
* **ЦАП:** 12-разрядные преобразователи.
* **Аналоговый тракт:** фильтры нижних частот (ФНЧ) Бесселя 2-го порядка для сглаживания ступенек квантования.
* **Изоляция:** оптическая развязка интерфейса RS232 для безопасной связи с ПК.
* **Интерфейс пользователя:** LCD-дисплей 16x2 и энкодер для навигации по меню.
* **Питание:** 5В.

## Программное обеспечение
* **Язык:** firmware - C (AVR-GCC), software - Go.
* **Загрузка данных:** Возможность загрузки тестовых отсчетов ЭКС во внутреннюю память через RS232.

## 📋 Соответствие стандартам
Имитатор спроектирован для обеспечения метрологических характеристик, необходимых при первичной и периодической поверке электрокардиографов.
